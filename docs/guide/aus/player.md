# AusJava 服务器游玩指南

欢迎来到 AusJava 服务器！本指南将帮助你快速了解服务器的特色功能和游玩方式。

## 服务器信息

### 基本信息

- **服务器地址**: `加群894835564获取，位于群公告内`
- **版本支持**: 1.19.4最佳

## 新手入门

### 初次进入

1. **出生点保护**
   - 出生点周围 15 格为保护区
   - 保护区内无法破坏方块
   - 稍微走远一点即可开始游戏

2. **基础生存**
   - 前期可以正常生存发展
   - 等有了一定规模可以发展科技
   - 服务器内会遇到一些怪物是英文名字，这是插件生成的默认怪物，比较强，没有装备最好不要碰。

3. **物品恢复**
   - 肝了很久的东西在下界掉岩浆了？
   - 没关系，你可以联系服主为你恢复物品栏一次

4. **死亡不掉落**
   - 只有主世界有死亡不掉落
   - 诸如下界，末地，地球高空轨道等**均没有**死亡不掉落，小心行事

### 重要提示

- **下界未开放** - 下界末地没有死亡不掉落，小心行事
- **主世界安全** - 在出生点石英建筑内你是安全的，不会被受到实体伤害
- **记录插件** - 服务器内有记录插件，所以不用担心家被人破坏，同时建议设置领地
- **地形修改** - 服务器内的地形是修改过的，所以与原版稍有偏差

## 核心功能

### 职业系统 (Jobs)

通过工作赚取金币，每个人最多可以同时加入三个职业。

#### 查看并加入职业

打开职业界面。

```
/jobs join
```

### 经济系统

#### 查看余额

```
/bal
```

#### 转账

```
/pay <玩家名> <金额>
```

### 商店系统

#### QuickShop (快捷商店)

玩家可以创建自己的商店：

1. 手持要出售的物品
2. 左键点击箱子创建商店
3. 输入价格
4. 完成创建

#### GlobalMarketChest (全球市场)

服务器内有全球市场，你可以使用命令`/gmc open simple`打开全球市场
- 浏览其他玩家的商品
- 出售自己的物品
- 购买需要的资源

### 传送系统

#### 基础传送

```
# 传送到玩家
/tpa <玩家名>

# 请求玩家传送过来
/tpahere <玩家名>

# 接受传送请求
/tpaaccept

# 拒绝传送请求
/tpadeny
```

### 领地系统 (Residence)

保护你的建筑不被破坏。

#### 创建领地

```
# 选择区域（使用木锄）
左键点击第一个点
右键点击第二个点

# 创建领地
/res create <领地名>

# 查看领地设置
/res set

# 删除领地
/res remove <领地名>
```

### Slimefun 粘液科技

服务器最大的科技系统，提供大量科技物品。

#### 入门指南

1. **获取粘液科技指南**
   ```
   /sf guide
   ```
   或使用命令 `/sf open_guide` 打开菜单


### mcMMO 技能系统

通过游戏行为提升技能等级，获得特殊能力。
例如你在摔倒的时候会有技能提升提示。
手持武器/工具时右键，此时攻击/挖掘会启用技能。

### 帮派系统 (SimpleClans)

创建或加入帮派

```
# 打开帮派系统
/clan
```

### 聊天展示物品

在聊天时发送`[i]`即可向大家展示手中物品。
发送`[inv]`展示背包。

### 系统商店

你可以通过金币在系统商店限量购买物品，也可以限量出售物品赚钱。
使用`/ultimateshop`打开系统商店.

## 服务器规则

### 基本规则

1. **禁止作弊** - 使用任何作弊客户端或矿透
2. **禁止恶意破坏** - 破坏他人建筑或领地
3. **禁止刷物品** - 利用 Bug 刷物品（生电除外）
4. **禁止辱骂** - 尊重其他玩家
5. **禁止广告** - 发送其他服务器广告

### 建筑规范

- 尊重他人的建筑和领地
- 建议圈领地保护自己的建筑

### 经济规范

- 公平交易，不欺诈
- 不恶意操控市场价格
- 不利用 Bug 获取金币

## 常见问题

### 如何开始赚钱？

1. 打开职业系统 `/jobs join`加入职业
2. 通过工作获得金币
3. 在市场出售物品
4. 开设自己的商店
5. 在系统商店(`/ultimateshop`)限量出售。

### 如何保护建筑？

使用 Residence 领地系统：
1. 用木锄选择区域
2. `/res create <名称>` 创建领地

### 如何开始科技？

1. 获取粘液科技指南 `/sf guide`
2. 制作增强工作台
3. 按照指南逐步解锁科技树

### 物品丢失怎么办？

1. 如果是被破坏，联系管理员查看记录
2. 如果是掉岩浆等意外，可以联系服主恢复一次
3. 建议物品存箱子内
