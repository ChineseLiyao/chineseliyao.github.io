# 介绍

欢迎使用 AusDoc 文档站点！

## 什么是 AusDoc？

AusDoc 是 AusJava 服务器的官方文档站点，使用 VitePress 构建。它提供了完整的服务器使用指南、API 文档和最佳实践。

## 特性

- **简单易用**：使用 Markdown 编写文档
- **快速响应**：基于 Vite 的极速开发体验
- **主题美观**：现代化的默认主题
- **搜索功能**：内置全文搜索
- **响应式设计**：完美支持移动端
