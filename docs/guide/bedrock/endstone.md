# Endstone 核心开服指南

Endstone 是一个新兴的 Minecraft 基岩版服务器核心，支持 Python 和 C++ 插件开发，为开发者提供了更灵活的插件开发体验。

## 什么是 Endstone？

Endstone 是一个基于 Bedrock Dedicated Server (BDS) 的插件加载器，它允许你使用 Python 或 C++ 编写插件来扩展服务器功能。

## 开服方法

### 环境要求
Python 3.10 或更高版本

### 下载Endstone
```bash
pip install endstone
```
这将使用pip包管理器自动下载endstone。

### 开服
切换到你的服务器目录。
运行以下命令:

```bash
endstone
```

首次启动会引导你下载原版服务端，输入Y回车继续。

### 编辑配置
与 BDS 原版配置 编辑方法相同。

### 加入服务器
启动服务器后，即可加入服务器游玩！
默认运行在端口19132上。

## 社区支持
Endstone 拥有活跃的社区和丰富的文档资源，你可以在 [Endstone 官方网站](https://endstone.dev/) 和 GitHub 仓库中找到更多帮助和支持。
你可以从[Bedrinth](https://bedrinth.com/?platform=endstone)内找到Endstone的插件。
